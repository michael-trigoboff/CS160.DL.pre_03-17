Assignment 1: Draw Your Initials

Write code that draws your initials (the first letter of each of your names) in a Python turtle graphics window. (If you have more than three names, you only need to do the first three.)

Feel free to experiment with different colors, dot sizes, and letter sizes. For available colors, see the Turtle Graphics Colors page in the Python Programming module.

Make sure to replace the value of the studentName variable with your own name. This will help me keep track of whose code I am running.

You can use any of the turtle graphics commands described in the Python Turtle Graphics Documentation.

Start with https://repl.it/@mtrigoboff/Assignment-01.

When you are done, upload the URL to your repl to Desire2Learn.

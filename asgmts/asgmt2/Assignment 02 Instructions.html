Assignment 02: Stack The Squares

Write code that draws a stack of five colorful squares in a Python turtle graphics window like this:

Feel free to experiment with different colors. For available colors, see the Turtle Graphics Colors page in the Python Programming module.

Make sure to replace the value of the studentName variable with your own name. This will help me keep track of whose code I am running.

You need to draw each square by calling the function named square, which is provided for you in the starter code. To draw the square, you need to use a loop in the function so that each time through the loop you draw one side of the square.

You can use any of the turtle graphics commands described in the Python Turtle Graphics Documentation.

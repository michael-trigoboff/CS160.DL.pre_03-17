Assignment 4: Click The Rectangles

Your code should create a Python turtle graphics window and draw at least five named rectangles in it. When the mouse is clicked in a rectangle, your code should print the rectangle's name using the Python print() function. When the mouse is clicked, but not inside any rectangle, your code should print the string, 'no hit'.

Start by specifying your rectangles as members of the list named rectangles. The data for each rectangle will be a list containing the following data:

[ name, bottomLeftX, bottomLeftY, width, height, outlineWidth, color]

When your window opens, it should go through the list of rectangles and draw them all in the window.

Assignment 6: Build a BMI Calculator

Task Description:

You will be building a Body Mass Index calculator like this:

You will build it using the pcc.gui package.

User Interface Items

The top three text fields ('Height (ft)', 'Height (in)', 'Weight (lbs)') are for user input. They can be clicked on and typed into (i.e. they are “enabled”). The fourth one ('BMI') is for showing the result of the Body Mass Index calculation. It cannot be typed into (i.e. it is “disabled”), which is indicated by its gray background. The text in it, however, can be selected and copied.

It is extremely important (meaning that you will lose significant points on your grade) for you to name your user interface items exactly as shown in the images. The required names are (in order, starting from the top):

Height (ft)
Height (in)
Weight (lbs)
BMI
Compute
Quit

Note the presence of one space before the left parenthesis in the first three names.

User Interface Behavior

The [Compute] button should only be enabled when there is valid numerical input in the three input text fields: 'Height (ft)', 'Height (in)', 'Weight (lbs)'. Otherwise it should be disabled.

When the [Compute] button is hilited, hitting Enter/Return on the keyboard should have the same effect as clicking the [Compute] button with the mouse.

Typing into any of the three input text fields should cause the BMI text field to become empty.

The formula for computing the Body Mass Index is:

bmi = 703 * weightInPounds / heightInInches2

(Note that Python has an exponentiation operator, **. You Can write x2 in Python like this: x**2)

The BMI text field should display the result of the calculation with two digits to the right of the decimal point. You should use Python’s format string feature to accomplish this:

bmiStr = f'{bmi:.2f}'

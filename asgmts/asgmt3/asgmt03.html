Assignment 3: Draw A Shape

Write code that responds to mouse clicks and the keyboard in a Python turtle graphics window. Your code should draw dots the mouse is clicked, and remember those places (as X, Y coordinates in a Python list) until the user types a 'd'. Then it should draw lines connecting those places to create a shape and fill that shape with the fill color.

Assignment 5: Overlapping Rectangles

Create a Python turtle graphics window. When you click the mouse in an empty space in that window, your code should draw a rectangle. The features of the rectangle should be randomly chosen like this:

width = r.randint(100, 200)
height = r.randint(100, 200)
penWidth = r.randint(1, 8)
outlineColor = r.choice(outlineColors)
fillColor = r.choice(fillColors)

The variables outlineColors and fillColors should be lists containing the names of colors coded as Python strings. You can use any colors you like from the Python Turtle Graphics Colors page in this course. Each list should contain at least five colors.

The rectangle should be centered around where the mouse was clicked. For instance, if the mouse was clicked at x == 300, y == 250, and the randomly chosen width and height were width == 180, height == 120, the corners of the rectangle would be:

corner x y
bottom left 210 190
bottom right 390 190
top left 210 310
top right 390 310

If you click on a rectangle, it should repaint itself. This will look like the rectangle has been "brought to the top," above all the rectangles that overlap it. In other words, the rectangles will behave like the windows in a graphical user interface.

Finally, if you type an e (for erase), your code should erase all the rectangles. It will do this by painting all of them with outline color and fill color set the same as the background color of the window.

For your code to do this, you will need to keep a list of all the current rectangles, and empty that list when you have erased them.

This will all be demonstrated in class.

You can use any of the turtle graphics commands described in the Python Turtle Graphics Documentation.
